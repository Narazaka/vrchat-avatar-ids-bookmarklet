VRChat Avatar Ids bookmarklet

source code

Create a new bookmark and paste the string above into the URL field. After logging in to VRChat Home, you can copy the list of IDs by executing the bookmark.
Paste the copied data into a text editor, etc., and select only the line with the ID you wish to load and paste it into the world text box.

新しくブックマークを作って、上の文字列をURL欄に貼り付けて下さい。VRChat Homeにログインしてからそのブックマークを実行するとID一覧がコピーできます。
コピーされたデータをテキストエディタなどにペーストし、ロードしたいIDの行のみを選んだものをワールドのテキストボックスにペーストして下さい。
